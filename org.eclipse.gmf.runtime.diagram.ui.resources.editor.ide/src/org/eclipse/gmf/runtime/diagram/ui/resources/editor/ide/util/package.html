This package contains utility classes used by this plugin which have IDE dependencies. For example, the {@link org.eclipse.gmf.runtime.diagram.ui.resources.editor.ide.util.IDEEditorFileCreator} IDEEditorFileCreator will take other operations taking place in the IDE in account when performing file modifications and the {@link org.eclipse.gmf.runtime.diagram.ui.resources.editor.ide.util.IDEEditorUtil IDEEditorUtil's} method of opening diagrams is dependent on having an IDE.
