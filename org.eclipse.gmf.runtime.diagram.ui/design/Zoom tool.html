Introduction

[back to top]

This feature introduces the ability to zoom in and out on the diagram surface using a tool to specify the context of where the zoom action will be centered on..

[back to top]

RATLC00532659 - Zoom tool

Requirements

[back to top]

- Should be able to zoom in on a specific element or set of elements.
- Ability to zoom-in and zoom-out.
- Integrated with existing zoom functionality via the toolbar and animated zoom.

Constraints and Limitations

[back to top]

A keyboard modifier is nescessary in order to access the zoom-out functionality since right mouse click is reservered for context menu invocation.

Zoom in on a point

[back to use cases]

The user wishes to zoom to a particular area of the diagram.

Preconditions:

  • Diagram is open.

Flow of events:

  1. User selects the zoom tool.
  2. User left mouse clicks on the diagram surface at a specific location

Postconditions:

  • Diagram zooms in by an incremental percentage with the point location staying at the center of the diagram view port.

Zoom out on a point

[back to use cases]

The user wishes to zoom out of a particular area of the diagram.

Preconditions:

  • Diagram is open.

Flow of events:

  1. User selects the zoom tool.
  2. User presses down the <SHIFT> key and keeps it depressed.
  3. User left mouse clicks on the diagram surface at a specific location.

Postconditions:

  • Diagram zooms out by an incremental percentage with the point location staying at the center of the diagram view port.

Zoom in to a rectangular area

[back to use cases]

The user wishes to zoom in to a particular rectangular area of the diagram.

Preconditions:

  • Diagram is open

Flow of events:

  1. User selects the zoom tool.
  2. User left mouse clicks on the diagram surface and drags the mouse to create a rectangular feedback selection box.
  3. User releases the mouse button after creating the selection rectangle.

Postconditions:

  • Diagram zooms in completely to the specified rectangle with the rectangle bounds filling out to the bounds of the view port.
